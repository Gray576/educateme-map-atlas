# A1 / A8 / A9 Comparison Pack

This export pack contains the minimum high-signal files for comparing:

- `A1` — AI Literacy Workshop
- `A8` — B2B Language Proficiency Audit
- `A9` — Integration Bundle (Sproochtest+)

Each product folder includes:

- `dashboard-ingestion.json`
  - safe fields
  - analyst-only fields
  - blocked fields
  - verification summary
- `commercial-validation.json`
  - validation velocity
  - demand pull
  - regulatory context
  - SEMrush-backed market evidence
- `claim-ledger.json`
  - verified / contradicted / blocked claim inventory
- `manual-review.md`
  - human review summary and current release logic
- `contradiction-log.json`
  - explicit contradictions and unresolved issues

Recommended reading order:

1. `dashboard-ingestion.json`
2. `commercial-validation.json`
3. `manual-review.md`
4. `claim-ledger.json`
5. `contradiction-log.json`

Not included on purpose:

- full `semrush-corpus`
- raw captures
- source ledgers
- batch manifests

Those remain in the main research run and are not needed for a focused 3-product comparison.
