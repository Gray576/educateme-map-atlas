## Report order per domain

For each chosen domain:

1. `Domain Overview`
2. `Organic Positions`
3. if the positions look relevant, keep the domain
4. if the positions are too broad or irrelevant, drop it and move on

## Filtering rule

Keep a domain only if its visible keyword family helps one of these:

- direct offer language
- problem language
- route / compliance proxy language

Drop it if it only contributes:

- generic broad education noise
- purely branded traffic
- unrelated language-learning surface
