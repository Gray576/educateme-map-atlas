# Multilingual Keyword Pairs for SEMrush

Цель этого списка: не искать "еще больше слов вообще", а расширять keyword universe по отдельным рыночным lane.

Как использовать:
- Берешь одну пару: `base keyword + language`.
- Ищешь ее в `Keyword Overview` и затем в `Keyword Magic Tool`.
- Если видишь живой cluster, снимаешь expansion уже вокруг него.
- Если видишь нули и мусорный хвост, дальше не расширяешь.

Базовое правило:
- Для `LUX` сначала смотри `FR`, `DE`, `LB`, потом `EN`.
- Для `global exam` сначала `EN`, потом нужный локальный язык.
- Для `Pflege / nurse` сначала `DE`.
- Для `compliance` сначала `FR + EN`, потом `DE`.

## Как интерпретировать язык потом по скриншоту

Если на скрине не видно явно выбранный язык, обычно можно восстановить его по самому query:

- `FR`: `cours`, `formation`, `apprendre`, `réussir`, `inscription`
- `DE`: `prüfung`, `kurs`, `lernen`, `schulung`, `weiterbildung`
- `LB`: `lëtzebuergesch`, `letzebuergesch`
- `EN`: `training`, `course`, `preparation`, `practice test`

Но лучшее правило дальше:
- в имени скриншота добавляй суффикс, например:
  - `mifid-training_EN.png`
  - `formation-compliance-luxembourg_FR.png`
  - `telc-b1-prufung_DE.png`
  - `sproochentest-letzebuergesch_LB.png`
